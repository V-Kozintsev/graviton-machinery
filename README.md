# GRAVITON Machinery

Адаптивный сайт-каталог строительной техники GRAVITON: первый экран, категории техники, карточки моделей, фильтры, детальная страница товара, заявка на КП, преимущества, этапы сделки и контакты.

Посмотреть сайт: https://v-kozintsev.github.io/graviton-machinery/

## Что использовалось

- React, TypeScript и CSS
- Vite для локального запуска и сборки
- React Router для страниц и чистых URL
- React Hook Form и Zod для формы заявки
- Embla Carousel для галереи товара
- GitHub Pages для публикации

## Локальный запуск

```bash
pnpm install
pnpm dev
```

## Сборка

```bash
pnpm build
```
